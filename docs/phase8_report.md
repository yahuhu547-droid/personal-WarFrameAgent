## 性能优化效果

### 缓存优化
- **场景**: 用户短时间内多次查询同一物品
- **效果**: 60秒内重复查询直接返回缓存，响应时间从 ~500ms 降至 <1ms
- **示例**: 查询"充沛"后立即查询"充沛散件"，复用缓存数据

### 限速保护
- **场景**: 批量查询、监控扫描
- **效果**: 自动控制请求频率，避免触发 API 限制
- **配置**: 每秒最多 3 次请求（可调整 `_rate_limit_delay`）
