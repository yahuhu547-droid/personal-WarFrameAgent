# Warframe 交易助手 Web UI 设计总结

## 项目概述

**项目名称**: Warframe 个人交易助手
**设计风格**: Tenno 科技终端
**设计师**: Claude (Frontend Design Specialist)
**完成时间**: 2026-04-30

## 设计成果

### 1. 视觉设计

#### 色彩方案
- **主色调**: 深空黑 (#070a14) + Tenno 金 (#d4a737) + 能量蓝 (#4a9eff)
- **辅助色**: 成功绿 (#4ade80)、警告橙 (#f59e0b)、错误红 (#ef4444)
- **渐变效果**: 金色渐变、蓝色渐变、背景渐变

#### 字体设计
- **标题字体**: Orbitron - 未来感强，适合标题
- **正文字体**: Rajdhani - 科技感，适合正文
- **数据字体**: JetBrains Mono - 清晰易读，适合价格数据

#### 视觉效果
- **扫描线效果**: 模拟终端扫描线
- **光效边框**: 金色/蓝色发光边框
- **几何装饰**: Warframe 风格的角落装饰
- **背景纹理**: 暗纹背景增加层次感

### 2. 交互设计

#### 动画效果
- **页面加载**: 元素逐个出现动画
- **消息动画**: 用户/Agent 消息从不同方向滑入
- **价格动画**: 价格上涨/下跌时的闪烁效果
- **按钮动画**: 悬停时的光效扫过
- **模态动画**: 弹窗缩放+淡入效果

#### 交互反馈
- **搜索建议**: 输入时实时下拉补全
- **键盘快捷键**: Enter 发送、Esc 关闭、Ctrl+K 聚焦
- **Toast 通知**: 页面内通知系统
- **加载状态**: Warframe 风格的点状加载
- **错误提示**: 友好的错误状态显示

### 3. 响应式设计

#### 断点设计
- **移动端**: < 640px
- **平板端**: 640px - 1024px
- **桌面端**: > 1024px
- **大屏幕**: > 1440px
- **超大屏幕**: > 1920px

#### 适配优化
- **触摸设备**: 增大点击区域，优化触摸反馈
- **安全区域**: 适配刘海屏等异形屏
- **滚动条**: 自定义滚动条样式
- **打印样式**: 支持页面打印

### 4. 无障碍设计

#### 可访问性
- **高对比度模式**: 支持高对比度显示
- **减少动画模式**: 尊重用户动画偏好
- **焦点样式**: 清晰的焦点指示
- **键盘导航**: 完整的键盘操作支持
- **屏幕阅读器**: 语义化 HTML 结构

## 技术实现

### CSS 架构
```
variables.css   - CSS 变量定义（颜色、字体、间距等）
animations.css  - 动画关键帧定义
style.css       - 主样式文件
responsive.css  - 响应式设计
```

### JavaScript 架构
```
app.js          - 主应用逻辑、API 调用、通知系统
chat.js         - 对话功能、消息管理
sidebar.js      - 侧边栏功能、收藏/提醒管理
chart.js        - 价格图表、统计信息
```

### 设计原则
1. **一致性**: 使用 CSS 变量确保样式统一
2. **可维护性**: 模块化 CSS 和 JavaScript
3. **性能**: 使用 CSS 动画而非 JavaScript
4. **无障碍**: 支持高对比度、减少动画模式
5. **响应式**: 适配各种设备尺寸

## 使用说明

### 访问页面
1. 启动 Web 服务器：`python start_web.py`
2. 浏览器访问：http://127.0.0.1:8000

### 首次访问
1. 显示欢迎弹窗
2. 选择游戏平台（PC/PS/Xbox/Switch）
3. 点击"开始使用"

### 主要功能
- **对话**: 输入物品名称或问题
- **搜索建议**: 输入时自动显示建议
- **收藏管理**: 在侧边栏查看和管理收藏
- **价格提醒**: 在侧边栏查看和管理提醒
- **价格图表**: 点击物品查看详情和图表
- **多物品对比**: 点击"对比物品"按钮

### 键盘快捷键
- `Enter`: 发送消息
- `Esc`: 关闭建议/面板
- `Ctrl+K`: 聚焦搜索框
