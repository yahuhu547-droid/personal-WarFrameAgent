# Warframe 交易助手 Web UI 美化实现报告

## 设计概述

### 设计理念
采用 **"Tenno 科技终端"** 设计概念，将 Warframe 交易助手打造成一个来自 Warframe 世界的高科技交易系统。

### 设计风格
- **主风格**：未来科幻工业风 + 军事终端感
- **色彩方案**：深空黑 + Tenno 金 + 能量蓝
- **字体**：Orbitron (标题) + Rajdhani (正文) + JetBrains Mono (数据)
- **特效**：扫描线、光效、几何装饰

## 设计亮点

### 1. Tenno 科技感
- **金色强调色**：使用 Warframe 标志性的 Tenno 金
- **几何装饰**：角落装饰线条，模拟游戏 UI
- **扫描线效果**：终端风格的扫描线叠加
- **光效边框**：金色/蓝色发光边框

### 2. 沉浸式体验
- **打字机效果**：Agent 消息逐字显示
- **加载动画**：Warframe 风格的点状加载
- **消息动画**：用户/Agent 消息从不同方向滑入
- **价格动画**：价格上涨/下跌时的闪烁效果

### 3. 交互反馈
- **按钮光效**：悬停时的光效扫过
- **焦点状态**：输入框焦点时的金色光晕
- **列表项悬停**：向右滑动并改变边框颜色
- **模态弹窗**：缩放+淡入动画

### 4. 信息层次
- **字体层次**：Orbitron (标题) → Rajdhani (正文) → JetBrains Mono (数据)
- **颜色层次**：主色 → 次色 → 辅助色
- **间距层次**：使用 CSS 变量统一管理
- **动画层次**：不同元素使用不同动画

## 技术实现

### CSS 架构
```
variables.css   - CSS 变量定义
animations.css  - 动画关键帧
style.css       - 主样式
responsive.css  - 响应式设计
```

### JavaScript 架构
```
app.js          - 主应用逻辑
chat.js         - 对话功能
sidebar.js      - 侧边栏功能
chart.js        - 图表功能
```

### 设计原则
1. **一致性**：使用 CSS 变量确保样式统一
2. **可维护性**：模块化 CSS 和 JavaScript
3. **性能**：使用 CSS 动画而非 JavaScript
4. **无障碍**：支持高对比度、减少动画模式
5. **响应式**：适配各种设备尺寸

## 使用说明

### 访问页面
1. 启动 Web 服务器：`python start_web.py`
2. 浏览器访问：http://127.0.0.1:8000

### 首次访问
1. 显示欢迎弹窗
2. 选择游戏平台（PC/PS/Xbox/Switch）
3. 点击"开始使用"

### 主要功能
- **对话**：输入物品名称或问题
- **搜索建议**：输入时自动显示建议
- **收藏管理**：在侧边栏查看和管理收藏
- **价格提醒**：在侧边栏查看和管理提醒
- **价格图表**：点击物品查看详情和图表
- **多物品对比**：点击"对比物品"按钮

### 键盘快捷键
- `Enter`：发送消息
- `Esc`：关闭建议/面板
- `Ctrl+K`：聚焦搜索框
